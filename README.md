# FlatMagic

A Figma plugin for enhancing your design workflow.

## Description
FlatMagic is a powerful Figma plugin that helps designers work more efficiently with their designs.

## Installation
1. Open Figma
2. Go to Plugins
3. Search for "FlatMagic"
4. Click "Install"

## Development
This plugin is built using:
- HTML
- JavaScript
- Figma Plugin API
